Drivers
-------

The MongoDB PHP Driver consists of the extension and library. 
While it is possible to use the extension alone, MongoDB recommends 
using both the extension and the library together.

.. card-group::
   :columns: 2

   .. card::
      :headline: Extension
      :url: https://www.mongodb.com/docs/
      :icon: /icons/driver-card-icon.svg
      :icon-alt: Driver card icon

      Provides a low-level API and mainly serves to integrate libmongoc and libbson with PHP.

   .. card::
      :headline: Library
      :url: https://www.mongodb.com/docs/
      :icon: /icons/driver-card-icon.svg
      :icon-alt: Driver card icon

      Provides a high-level API for working with MongoDB databases consistent with other MongoDB language drivers.
